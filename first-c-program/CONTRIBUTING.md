# 🤝 Contributing to DO YOU LOVE ME?

First off, thanks for taking the time to contribute! ❤️

## How Can I Contribute?

### 🐛 Reporting Bugs

- Check if the bug has already been reported in [Issues](https://github.com/hexa/do-you-love-me/issues)
- If not, open a new issue with:
  - Clear title and description
  - Steps to reproduce
  - Expected vs actual behavior
  - Screenshots if possible

### 💡 Suggesting Enhancements

- Open an issue with the tag `enhancement`
- Describe your idea clearly
- Explain why it would be valuable

### 🔧 Pull Requests

1. Fork the repo
2. Create a new branch: `git checkout -b feature-name`
3. Make your changes
4. Test your changes
5. Commit: `git commit -m "Description"`
6. Push: `git push origin feature-name`
7. Open a Pull Request

### 📝 Code Style

- Use consistent indentation (4 spaces)
- Add comments for complex logic
- Keep functions simple and focused
- Test your code before submitting

### 🎯 Good First Issues

Looking for something easy? Try:
- Fixing the `scanf` buffer issue
- Fixing the "aline" typo
- Adding more responses
- Improving error handling

## 🚀 Development Process

1. Clone the repo
2. Make your changes
3. Test thoroughly
4. Submit PR

## 💬 Questions?

Open an issue or reach out to Hexa!
